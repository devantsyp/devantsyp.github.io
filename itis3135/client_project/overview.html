UNCC Breaking Project Overview

Project Name: UNC Charlotte Breakdancing Club Web Hub

Intended Purpose: This web application aims to promote the UNC Charlotte breakdancing club, showcase its members and events, and provide useful resources for current UNCC students who are interested in joining or learning about breakdancing. The platform will help improve communication, increase visibility on campus, and serve as an interactive information hub.

Client Information: The client for this project is Mitch, a student, who is organizing a breakdancing club at the UNCC campus. A valid email address is unccbreaking@gmail.com.

Site map

  1. Home Page
    • Purpose: Introduce the UNCC Breaking club and invite users to explore and join.
    • Audience: General public, potential members..
    • Content: Banner with logo, mission statement, welcome message, navigation links. Footer includes HTML/CSS validation links.
  2. About Page
    • Purpose: Share the club’s history and mission.
    • Audience: General public, new/prospective members.
    • Content: Brief overview of the club, member profiles/bios, accordion expands/collapses. Embedded Google Maps location.
  3. Events Page
    • Purpose: Promote practices, performances, future meetups, local events.
    • Audience: Members, interested students.
    • Content: Basic calendar or list format which includes practice times. Special sessions and workshops. Will have an interest form (e.g., “Let me know about future events!”). Includes a submit button.
  4. Gallery Page
    • Purpose: Display media from practices or battles.
    • Audience: Anyone interested and browsing the site.
    • Content: Selection of photos and/or clips.
  5. Learn Page
    • Purpose: Provide basic info about breakdancing, beginner tips, and how to get started.
    • Audience: Students interested in trying it out.
    • Content: How to get started” section. Basic terminology or style overview. Embedded video or link to Youtube resources. Collapsible FAQ or guide steps using jQuery/JS.
  6. Contact Page
    • Purpose: Lets users get in touch with the club.
    • Audience: Interested students, collaborators.
    • Content: Contact form, social media links, club email. Includes form fields, submit button

Dynamic functionality

  • Gallery page includes a gallery which allows the user to cycle through the images using navigation arrows. Inspired by National Geographic.
  • Learning Page has buttons to toggle between tutorials (e.g., "Basic Moves," "Power Moves," "Footwork") without reloading the page. Inspired by Apple.
  • The header has a button which enables the navbar dropdown menu and which is also animated.
  • Instagram posts embedded into the site across various pages.
  • Media queries for support across various resolutions.
